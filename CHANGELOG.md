## [0.1.1] - 2025-08-20

### 新增

- 添加 GitHub Issue 模板，包括错误报告和功能请求模板
- 创建项目基础架构
- 实现Room数据库存储月经记录
- 添加日历界面，支持月经周期可视化
- 添加记录列表界面，支持查看和管理历史记录
- 添加快速添加界面，支持快速记录症状
- 添加统计分析界面，提供基本的周期统计
